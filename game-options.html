Reference Guide

Game Options

Kairos has two layers of configuration: the BGA table settings when creating a game, and the in-game Advanced options modal shown to the table creator before play begins.

🎲

BGA Table Settings

When creating the table

These options appear in the BGA table creation screen before players join. They control the overall game mode and whether AI players are included.

⚙️ Game Mode Choice
Basic

Standard rules with default settings. Recommended for new players — jump straight in with no extra decisions.

Advanced

The table creator gets an in-game options modal before play starts, allowing full control over difficulty, game length, and special features.

Default: Basic
🤖 Automa (AI) Players Advanced Only

Add AI-controlled players to fill empty seats. This option only appears when Game Mode is set to Advanced. The number of agents each human player controls depends on the total player count (humans + automa), so this must be decided before the game starts.

Total PlayersAgents per Human
2 players6 agents
3 players4 agents
4 players3 agents
AutoBeta

Standard AI difficulty — good for learning and casual play.

AutoAlpha ★

Harder AI — makes smarter decisions and is more aggressive.

Default: None (all human players)
🛠️

In-Game Options (Advanced Mode)

Before play begins

When Game Mode is set to Advanced, the table creator sees an options modal after all players have joined but before team setup. These settings shape the game's difficulty, length, and special features.

🎯 Difficulty Choice
Easy

Crystal Light gets an extra +1 bonus. More forgiving for new teams learning the mechanics.

Medium

Standard rules as designed. The intended Kairos experience.

Hard

Kronos darkness gets extra power. For experienced teams seeking a real challenge.

Default: Medium
📏 Game Length Choice

Adjusts the number of rounds before the Singularity arrives. The base round count depends on how many players are in the game.

Short

Base rounds minus 2. Fast-paced — every turn counts.

Standard

Base rounds. The designed game length for the player count.

Long

Base rounds plus 2. More time to explore and strategise.

Default: Standard
🥚 Easter Egg Crystal Toggle

One crystal on the board is secretly the Easter Egg. If your team finds it and deposits it at HQ, you earn a special bonus. Adds a layer of exploration and discovery to the crystal-hunting mechanic.

Default: OFF
🦕 Time-Travelling Dinosaurs Toggle

Dinosaurs roam the time field, displaced from their own era! Send them back to earn rewards. A fun optional mechanic that adds side objectives and extra strategic choices during agent turns.

Default: OFF
🛠️ Debug Mode Toggle

Shows the debug panel for testing and development purposes. Not needed for normal gameplay — intended for game testing and bug reporting.

Default: OFF
💡 First Game Recommendation

For your first game, use Basic mode. This gives you standard rules with sensible defaults so you can focus on learning the core mechanics. Once you're comfortable, switch to Advanced to unlock difficulty tuning, game length control, and the optional features like Easter Egg and Dinosaurs.
